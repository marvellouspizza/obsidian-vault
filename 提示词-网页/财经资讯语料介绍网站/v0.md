## 项目概述

创建一个专业的财经资讯实时发布与管理平台，聚合15家主流财经媒体，提供全方位的财经资讯展示、筛选、搜索和导出功能。

## 数据库设计

### 表结构：`finance`

```sql
CREATE TABLE finance (
  id SERIAL PRIMARY KEY,
  source TEXT NOT NULL,                    -- 消息来源
  timestamp TIMESTAMPTZ DEFAULT NOW(),     -- 发布时间
  content JSONB NOT NULL,                  -- 内容段落（JSONB数组）
  images JSONB,                            -- 配图URL（JSONB数组）
  table_data JSONB,                        -- 表格数据（可选）
  tags JSONB NOT NULL DEFAULT '[]'::JSONB, -- 标签数组
  likes INTEGER DEFAULT 0,                 -- 点赞数
  bookmarked BOOLEAN DEFAULT false,        -- 是否收藏
  created_at TIMESTAMPTZ DEFAULT NOW()
);

-- 索引
CREATE INDEX idx_finance_timestamp ON finance(timestamp DESC);
CREATE INDEX idx_finance_source ON finance(source);

-- 启用 RLS
ALTER TABLE finance ENABLE ROW LEVEL SECURITY;
CREATE POLICY "Allow all access to finance" ON finance FOR ALL USING (true);
```

### 2. Supabase 客户端配置 (`lib/supabase.js`)

**关键要点**：
- 使用 `@supabase/ssr` 的 `createBrowserClient`
- 实现单例模式，避免重复创建客户端
- 支持浏览器环境的 `window.__NEXT_PUBLIC_*` 变量读取
- 未配置时返回 mock 客户端，避免应用崩溃

```javascript
import { createBrowserClient } from "@supabase/ssr"

let client

export function getSupabaseClient() {
  if (!client) {
    const supabaseUrl = typeof window !== "undefined"
      ? window.__NEXT_PUBLIC_SUPABASE_URL__ || process.env.NEXT_PUBLIC_SUPABASE_URL
      : process.env.NEXT_PUBLIC_SUPABASE_URL

    const supabaseAnonKey = typeof window !== "undefined"
      ? window.__NEXT_PUBLIC_SUPABASE_ANON_KEY__ || process.env.NEXT_PUBLIC_SUPABASE_ANON_KEY
      : process.env.NEXT_PUBLIC_SUPABASE_ANON_KEY

    if (!supabaseUrl || !supabaseAnonKey) {
      console.warn("Supabase not configured")
      return { /* mock client */ }
    }

    client = createBrowserClient(supabaseUrl, supabaseAnonKey)
  }
  return client
}
```

## 交互细节

### 筛选逻辑
1. **来源筛选**：多选模式，点击切换选中状态
2. **关键词搜索**：匹配 `content` 段落和 `tags` 标签
3. **日期范围**：支持开始日期、结束日期或范围筛选
4. **实时更新**：使用 `useEffect` 监听筛选条件变化

### 导出功能
- 批量导出：XLSX（Excel）、PDF、DOCX（Word）、MD（Markdown）
- 单篇导出：点击卡片导出按钮
- 分享功能：复制带筛选参数的URL

- 选中来源：琥珀金色背景 + 深色文字
- 收藏状态：星标图标填充琥珀色
