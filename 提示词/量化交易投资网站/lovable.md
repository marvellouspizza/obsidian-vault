# 智驭量化 (AlphaControl) 官网 - 完整实现提示词

## 项目概述

A股量化私募投资公司官方网站，体现"严谨数学逻辑"与"前沿AI技术"的量化投资展示平台。

## 视觉设计系统

### 配色方案 (HSL格式)

背景色: 深蓝灰 hsl(222, 47%, 8%)
前景色: 冷白色 hsl(210, 40%, 98%)
主色调: 电光青 hsl(186, 100%, 50%)
次要色: 深板岩蓝 hsl(217, 33%, 17%)
强调色: 数据绿 hsl(142, 76%, 45%)
警告色: 科技橙 hsl(25, 95%, 53%)
图表色: 青色/紫色/橙色/绿色渐变系列

### 字体系统

- **中文**: Noto Sans SC (思源黑体)
- **代码/数据**: JetBrains Mono (等宽字体)

### 特效样式

- glow-border: 发光边框效果
- data-grid: 网格背景纹理
- terminal-text: 终端风格文字
- scan-line: 扫描线动画
- pulse-dot / flow-line: 脉冲与流动动画

## 页面结构

### 1. 导航栏 (Header)

- Logo + 公司名称
- 导航项: 投资策略 | 技术底座 | 业绩表现 | 关于我们
- 响应式移动端菜单
- 滚动时背景模糊效果

### 2. 首屏 (HeroSection)

- **主标题**: 数据驱动前瞻，算法定义未来
- **副标题**: 深耕A股微观结构，以科学视角捕捉非对称盈利机会
- **核心卖点徽章**: [高频数据接入] [机器学习建模] [分布式回测集群]
- **统计数据栏**: 管理规模 50亿+ | 策略运行 1000+ 天 | 年化超额 15%+
- 动态背景网格 + 浮动光点效果

### 3. 策略模拟计算器 (AlphaCalculator)

- **双滑块交互**:
    - 预期超额收益 (0-30%)
    - 风险偏好 (1-10)
- **实时计算展示**:
    - 动态净值曲线图表 (recharts ComposedChart)
    - 夏普比率 / 最大回撤 / 期末收益
- 模拟数据基于参数动态生成

### 4. 策略归因切换 (StrategyToggle)

- **三大策略按钮**: 指数增强 | 市场中性 | CTA趋势
- **策略详情卡片**:
    - 策略描述
    - 核心指标 (年化收益/夏普/回撤/换手率)
    - 收益曲线图表 (AreaChart)
- 切换时平滑过渡动画 (AnimatePresence)

### 5. 因子挖掘流程 (FactorPipeline)

- **四阶段流程图**:
    1. 原始行情数据 → 2. 特征工程 → 3. 模型训练 → 4. 信号输出
- **悬停弹窗**: 显示每个节点的技术参数
- SVG连接线 + 流动动画
- 移动端自适应卡片布局

### 6. 业绩指标看板 (PerformanceBoard)

- **核心指标卡片**: 年化收益 | 夏普比率 | 最大回撤
- **图表展示**:
    - 累计净值走势 (AreaChart)
    - 年度收益对比 (BarChart)
- Tooltip悬停显示具体数值

### 7. 页脚 (Footer)

- 公司简介
- 快速链接
- 联系方式
- 版权信息 + 风险提示

## 技术栈

- React 18 + TypeScript + Vite
- Tailwind CSS + shadcn/ui
- Framer Motion (动画)
- Recharts (图表)
- Lucide React (图标)

## 关键交互

1. 所有数值组件支持 Tooltip
2. 策略切换无刷新平滑过渡
3. 滑块实时计算反馈
4. 响应式适配桌面/移动端
