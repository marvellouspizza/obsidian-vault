# AlphaMind 量化交易平台 - 网站设计提示词


[https://v0-alpha-mind-quant-landing-page.vercel.app/](https://v0-alpha-mind-quant-landing-page.vercel.app/)
## 项目概述
为AlphaMind（专注A股市场的量化交易公司）创建一个专业的单页落地页网站，展示公司的量化交易策略、技术优势、实盘业绩和团队实力。

## 设计要求

### 视觉风格
- **主题色调**：深色主题（深海军蓝背景 #0a192f）
- **强调色**：
  - 主色：霓虹青色（oklch(0.7 0.18 195)）用于品牌和交互元素
  - 辅助色：翠绿色（oklch(0.68 0.16 160)）用于数据和图表
- **字体**：
  - 标题/正文：Inter 无衬线字体
  - 数字/代码：Roboto Mono 等宽字体
- **设计语言**：
  - 玻璃态效果（glassmorphism）
  - 柔和阴影和模糊背景
  - 圆角 12px（--radius: 0.75rem）
  - 悬浮动画和过渡效果

## 技术实现

### 技术栈
- **框架**：Next.js 16 (App Router)
- **样式**：Tailwind CSS v4
- **组件库**：shadcn/ui
- **图表**：Recharts
- **图标**：Lucide React
- **字体**：Inter + Roboto Mono (Google Fonts)

### 组件结构
\`\`\`
app/
├── page.tsx (主页面，组装所有section)
├── layout.tsx (根布局，配置字体和元数据)
└── globals.css (全局样式，CSS变量定义)

components/
├── navbar.tsx (导航栏，含滚动检测和移动端菜单)
├── hero-section.tsx (首屏，含数字动画)
├── strategies-section.tsx (策略卡片，含hover效果)
├── technology-section.tsx (技术特性)
├── performance-section.tsx (业绩图表，含时间切换)
├── research-workflow.tsx (研究流程)
├── risk-management-section.tsx (风险管理)
├── about-section.tsx (关于我们)
├── alpha-calculator.tsx (收益计算器，含滑块交互)
├── footer.tsx (页脚)
└── logo-icon.tsx (Logo图标组件，DNA螺旋+烛台融合设计)
\`\`\`

### 交互特性
1. **数字动画**：Hero区关键指标从0增长到目标值（2秒）
2. **卡片悬浮**：策略卡片hover时向上移动8px并改变边框颜色
3. **3D变换**：卡片图标hover时旋转5度并放大
4. **滚动检测**：导航栏根据滚动位置改变背景透明度
5. **时间范围切换**：业绩图表支持全部/1年/3年视图切换
6. **实时计算**：Alpha计算器滑块变化时立即更新结果
7. **链接下划线动画**：导航链接hover时下划线从左向右展开

## 内容规范

### 文案风格
- 专业、数据驱动、理性
- 使用金融量化术语（alpha、beta、夏普比率、VaR等）
- 强调合规性和风险提示
- 中文为主，专业术语保留英文

### 数据展示
- 所有百分比保留一位小数
- 货币金额使用中文单位（亿、万）
- 比率数值保留两位小数
- 使用等宽字体（font-mono）展示数字

### 响应式设计
- 移动端：单列布局，汉堡菜单
- 平板：两列网格布局
- 桌面：三列或四列网格布局
- 断点：sm(640px), md(768px), lg(1024px)
